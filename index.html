Scalajp Github Pagesへようこそ

このページは、ScalaJPのGithub上のサイトです。ScalaJP公式サイトは主に、サイトへの「入り口」および、 公式的なリンク集に近いものを提供して、このサイト上には、翻訳記事等を掲載していきます。現時点では、まだ検討段階ですが、記事だけでなく、実際のプロダクト (Specs2など)をforkして、コミュニティベースでいじってみる、といった事もできると面白いかもしれません。

現在のステータス

現在、ScalaJP Wikiがあります。 記事にするまでも無い小ネタ(Tipsなど)や、執筆途中の記事などどんどん編集してもらってかまいません(Githubの アカウントがあれば誰でも書き込めるようになっています)。

翻訳記事については、ScalaJPのorganizationにあるリポジトリ が、ScalaJPに移管されたものになっています。
